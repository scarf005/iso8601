# ISO 8601

[![](demo/page.png)][pages]

datetime display in
[ISO 8601](https://www.iso.org/iso-8601-date-and-time-format.html)

[Visit the page][pages]

## How to run

```sh
git clone https://github.com/scarf005/iso8601.git
cd iso8601
deno task dev
```

### How to test

```sh
deno test
```
